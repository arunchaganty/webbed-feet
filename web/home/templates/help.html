Introduction

Webbed Feet is simply an online judge that will let you upload your bots, and pit them against others; it was created to help give you a better idea of the competition, and to help you find deficiencies in your bots so that you may create even better ones. Webbed Feet isn't particular to any game as such, but this help document will describe what needs to be done to use it with Desdemona - an Othello game framework. Again, you can find Desdemona here. Documentation here.

  • Creating a Bot
  • Submitting a Bot

Creating a Bot

Desdemona has a plugin architecture where your code is compiled to a shared library (.so). You don't have to worry about how it handles the libraries, but you can check this HOWTO if you're curious. However, because Desdemona expects a library, it must be compiled specially, using the following flags for gcc, -fPIC and -shared.

Writing Code

Feel free to write code however you like. You must provide 2 functions though, createBot and destroyBot, which must be placed inside an extern "C" block. Check the examples to see what needs to be done.

A couple of simple functions have been provided for your convenience to find the valid moves, etc. in a map. Explore the header files provided with Desdemona to get a better idea of what you can use.

Compiling Code

A template setup has already been created for you in /bots/MyBot. In the Makefile, there are two important variables that you might want to modify, $BOT and $OBJS. If you'd like to rename your bot from MyBot to FooBot you should change the $BOT variable - however, remember that you will also need an appropriate FooBot.cpp. If you'd like to add more files and break up your code, say file1.cpp, file2.cpp, etc., just add file1.o, file2.o, etc. to $OBJS (the .o is important). Also, the output library must be called bot.so for any submission.

Creating a Submission

Webbed-Feet expects your submission to be a single .zip file, containing all the files you require and the appropriate makefile. The template setup will automatically do this for you; just type "make submission", which will produce a submission.zip that you can submit.

Submitting a Bot

Once you've gotten a .zip file as mentioned above, you can submit it in the Manage Submissions section. Enter in a name to help yourself identify what's special about the bot you've uploaded. I highly recommend that you test out your bot against your own first; several times.

  • Webbed Feet will play the bot with the least number of runs against a randomly selected bot, and continue doing this.
  • Your bots will only be played against others. If you want to check how they play against your own bots - do it locally.
  • If your bot causes some exception, either a time out, or a crash more than 3 times in a row, it will be deactivated.
  • You are allowed at most 3 active bots at any point in time.
